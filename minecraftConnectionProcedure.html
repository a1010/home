＜クライアント手順＞

  1. 公式のminecraftを一度起動する
    1. ver1.18.2を起動
    2. 起動出来たらゲームを閉じる
  2. Forgeのminecraftをインストール
    1. URLからダウンロード
    2. ファイルをダブルクリックしてインストーラーを起動
    3. Install client を選択し、OKを押下
      4. ※インストーラーを起動できない場合、java17をURLからインストール
  3. Forgeのminecraftを一度起動する
    1. 起動構成の変更
    2. 「…」→「編集」
    3. 「ゲームディレクトリ」を任意のフォルダに変更
    4. 起動構成の設定を閉じ、「プレイ」を押下
    5. 起動出来たらゲームを閉じる
  4. MODの配置
    1. 3．で指定した「ゲームディレクトリ」を開く
    2. mods フォルダを開く
    3. MOD Links の for Client から MOD をダウンロードし、mods フォルダに配置
  5. ゲームの起動
    1. 「プレイ」を押下
  6. サーバーへの接続
    1. 「マルチプレイ」を押下
    2. 「サーバーを追加」を押下
    3. サーバーアドレス：　に設定
    4. 「完了」を押下
    5. サーバーを選択し、「サーバーに接続」を押下
